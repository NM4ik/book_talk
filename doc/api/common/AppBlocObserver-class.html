AppBlocObserver

AppBlocObserver class final

Custom BlocObserver implementation to log BLoC transitions and errors.

AppBlocObserver monitors state transitions and errors in all BLoC instances and logs them via the provided AppLogger. This is useful for tracking BLoC state changes and diagnosing errors during development.
